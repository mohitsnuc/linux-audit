**Linux Overall Security auditor**
##Overview##
This has been specifically desgined for linux systems. This extracts core OS-level configurations using bash and then pipelined it with a python evaluation engine to flag vulnerabilities.

Features:
1. Linux evaluation: Looks for world writable files innthe system and then checks critical system file permissions
2. JSON: the raw data searched and obtained is then converted into a JSON file for easy pipelining and understanding
3. Python risk assesment: This reads the JSON file generated by the shell script and then evaluated metrics and throws clear warnings and alerts.

Prerequisities:
1. Linux Enviromnent
2. Python 3.x
3. basic linux utilities (find,wc,stat)
